## 1. Product List Display
- Show products in **List view** (table format)  
- Show products in **Card view** (grid format)  
- **Toggle** between List and Card views  

---

## 2. Search (with Debounce)
- Search products by **name** in real-time  
- **Debounce** set to **500ms** to optimize search performance  

---

## 3. Add & Edit Product
Form includes:
- **Name** (required)  
- **Price** (number, required)  
- **Category** (required)  
- **Stock** (number)  
- **Description** (optional)  

**Features:**
- Basic form validation (shows error messages if invalid)  
- Update state **in memory** (no backend required)  

---

## 4. Pagination
- Navigate through multiple pages of products  
- Display limited products per page for better readability
